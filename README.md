# 🚀 Quick Start

### 1. Prerequisites
This project requires the **backend** to be running. Make sure you have set up:
👉 [hotel-booking-site](https://github.com/ron-don-don/hotel-booking-site)

### 2. Start the Server
Run this command in your terminal:
```bash
docker compose up -d --build
```

### 3. Access the App
The website is available at:  
**[https://localhost](https://localhost)**
